# Eren Avunc - Kişisel CV Sitesi

Bu repo, kişisel özgeçmişimi (CV) ve portföyümü içeren basit bir web sitesini barındırır.  
Site, HTML ve CSS kullanılarak hazırlanmıştır ve GitHub Pages üzerinden yayınlanmaktadır.

## Özellikler
- Mobil uyumlu tasarım (responsive design)
- Basit ve temiz arayüz
- Yazdırma için özel `@media print` ayarları (eklenecek)
- İndirilebilir CV butonu (eklenecek)

## Kullanım
Bu proje yalnızca kişisel CV sunumu amacıyla hazırlanmıştır.  
Kodun görüntülenmesi serbesttir, ancak kullanımı veya değiştirilmesi yasaktır.
